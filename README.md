# td-sustainable-ai-plugin
A browser extension that can reduce inefficient AI prompting

To run: 
run backend first with the following: 
cd backend 
node server.js 

may need to install 
npm init -y  
npm install express cors          

build front end: 
npm build 

then go to chrome and add extension via "load unpacked" and pick the extension folder. 
Enjoy:)
